# Быстрый старт

## Установка

1. Установите Python 3.10 или выше

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Установите Playwright (для будущей интеграции с Dzen):
```bash
playwright install
```

## Первый запуск

1. Запустите приложение:
```bash
python main.py
```

2. В открывшемся окне:
   - Перейдите на вкладку "Платформы"
   - Введите ваш Gemini API ключ
   - Нажмите "Сохранить API Key"

3. Перейдите на вкладку "Цели":
   - Настройте предпочитаемые темы
   - Настройте частоту публикаций
   - Сохраните цели

4. Вернитесь на "Dashboard":
   - Нажмите "Запустить" для начала работы

## Структура данных

При первом запуске будут созданы следующие директории:
- `data/` - база данных и конфигурация
- `logs/` - логи работы системы
- `sessions/` - сессии браузера (для Dzen)

## Конфигурация

Все настройки хранятся в `data/settings.json` и `data/goals.json`.

Токены платформ хранятся в зашифрованном виде в `data/tokens.encrypted`.

## Следующие шаги

После базовой настройки можно:
1. Подключить платформы (VK, Telegram, Dzen) - в разработке
2. Настроить более детальные цели
3. Настроить планировщик публикаций
