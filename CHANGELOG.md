# Changelog

## [1.0.0] - Полный релиз

### Добавлено

#### Платформы (Полная реализация)
- ✅ VK интеграция
  - OAuth авторизация через access token
  - Выбор группы из списка администраторских
  - Публикация постов на стену группы
  - Проверка прав администратора
  - Сохранение токенов в зашифрованном виде

- ✅ Telegram интеграция
  - Авторизация через bot token
  - Поддержка каналов и чатов
  - Публикация сообщений
  - Публикация тредов (серия сообщений)
  - Проверка прав администратора бота

- ✅ Dzen интеграция
  - Браузерная авторизация через Playwright
  - Сохранение сессии для повторного использования
  - Публикация статей
  - Автоматическое заполнение формы редактора

#### UI улучшения
- ✅ Диалог выбора группы VK
- ✅ Индикация статуса подключения платформ
- ✅ Сообщения об успехе/ошибках
- ✅ Обновление статуса в реальном времени

#### T5 Улучшения (из предыдущих версий)
- ✅ Internal Triggers - автономная воля
- ✅ Memory Pressure - активная память
- ✅ Meta-Critic - самокоррекция
- ✅ Self-Explanation Mode
- ✅ Personality Drift

### Изменено

- Publisher агент теперь использует реальные платформы вместо симуляции
- Entity интегрирует PlatformManager
- UI полностью функционален для подключения платформ

### Исправлено

- Ошибка импорта `Optional` в `utils/helpers.py`
- Дублирование полей в UI

### Технические детали

- Все платформы следуют единому интерфейсу `BasePlatform`
- Токены хранятся через `TokenStorage` с шифрованием
- Асинхронная обработка всех операций платформ
- Обработка ошибок и retry логика
