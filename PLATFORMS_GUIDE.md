# Руководство по подключению платформ

## VK

### Получение Access Token

1. Перейдите на https://oauth.vk.com/authorize?client_id=YOUR_APP_ID&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=wall,groups&response_type=token&v=5.154
2. Замените `YOUR_APP_ID` на ID вашего приложения VK
3. Разрешите доступ к стене и группам
4. После авторизации скопируйте `access_token` из URL
5. Вставьте токен в поле "Access Token" в приложении
6. Нажмите "Подключить VK"
7. Выберите группу из списка (будут показаны только группы, где вы админ)

### Требования

- У вас должны быть права администратора в группе
- Приложение должно иметь доступ к `wall` и `groups`

## Telegram

### Получение Bot Token

1. Найдите @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен (формат: `123456:ABC-DEF...`)
5. Вставьте токен в поле "Bot Token"

### Получение Chat ID

Для канала:
1. Создайте канал или используйте существующий
2. Добавьте бота как администратора канала
3. Получите Chat ID:
   - Для публичного канала: используйте `@channel_name`
   - Для приватного канала: используйте формат `-1001234567890`

Для получения числового ID приватного канала:
1. Перешлите сообщение из канала в @userinfobot
2. Используйте полученный ID (с минусом)

### Настройка

1. Вставьте Bot Token в поле "Bot Token"
2. Вставьте Chat ID в поле "Chat ID"
3. Нажмите "Подключить Telegram"
4. Убедитесь, что бот является администратором указанного чата

## Dzen

### Авторизация

1. Нажмите "Подключить Dzen"
2. Откроется окно браузера
3. Войдите в свой аккаунт Яндекс/Dzen
4. После успешной авторизации окно можно закрыть
5. Сессия сохранится автоматически

### Особенности

- Авторизация через браузер (Playwright)
- Сессия сохраняется локально
- Не требует токенов или ключей API
- Поддерживает публикацию статей

## Использование

После подключения платформ:

1. Система автоматически будет использовать подключенные платформы
2. При создании контента он будет опубликован на все подключенные платформы
3. Статус подключения можно проверить на вкладке "Платформы"

## Безопасность

Все токены и сессии хранятся локально в зашифрованном виде:
- VK tokens: `data/tokens.encrypted`
- Telegram tokens: `data/tokens.encrypted`
- Dzen session: `sessions/dzen/`
